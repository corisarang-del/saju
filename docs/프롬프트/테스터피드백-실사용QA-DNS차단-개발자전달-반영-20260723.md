# 테스터 피드백 실사용QA DNS차단 개발자전달 반영

## 사용자 요청
- 사용자가 `docs/pm/테스터피드백-실사용QA-DNS차단-개발자전달-20260723.md` 문서를 전달했다.
- 핵심 내용은 실제 Supabase DB 기반 API QA가 DNS 차단으로 미완료이므로 production 배포 승인으로 보면 안 된다는 것이다.

## 처리 방향
- Supabase project ref, 로컬 env, Supabase CLI link 상태를 먼저 확인했다.
- 코드/빌드 게이트와 실제 API QA 게이트가 섞이지 않도록 release script를 분리했다.
- production 승인용 `release:gate`에는 실제 API QA를 포함해, DNS/API QA 미완료가 blocker로 남도록 했다.
