# 월간사주 적용 요청

사용자는 `imgompanda/monthlysaju-opensource` 원본을 현재 워크스페이스에 가져오고, `월간사주`라는 이름의 사주 기반 라이프 에이전트 웹앱으로 업그레이드해달라고 요청했다.

핵심 요구:
- pnpm만 사용한다.
- TDD 방식으로 구현한다.
- 문서와 화면 명세를 먼저 정리한다.
- 소연 캐릭터를 하나로 리브랜딩한다.
- 캐릭터 선택 후 선택한 캐릭터가 오늘의 에이전트 피드와 대화를 운영한다.
- 오늘의 행동카드, 아침/점심/저녁 타임라인, 기억 기반 대화, 월간 리포트, 유료 종합 사주 백서를 포함한다.
- 이미지는 화사하고 세련된 고급 웹툰 상담가풍으로 생성 후보를 만든다.
