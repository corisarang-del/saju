## 전달 문서 핵심
- Paddle signed webhook 정상 서명 경로가 운영에서 401로 실패한다.
- production env gate가 `GOOGLE_VERTEX_RUNTIME_AUTH` 누락으로 실패한다.
- `/api/saju/preview`, `/api/saju/analyze`, `/api/saju/compatibility`, `/api/saju/chat`이 invalid JSON/body에 500을 반환한다.
- CSP `unsafe-inline` 잔여는 report-only와 nonce/hash 전략 검토가 필요하다.

## 처리 방향
- 코드로 닫을 수 있는 invalid JSON 500은 공통 helper와 회귀 테스트로 즉시 수정한다.
- Vercel production의 `GOOGLE_VERTEX_RUNTIME_AUTH`는 `vercel-oidc`로 등록한다.
- Paddle signed webhook은 실제 Paddle product/price id와 webhook secret이 없으면 QA가 차단되므로 남은 운영 작업으로 기록한다.
