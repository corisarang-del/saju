# 채팅 분석 응답 중간 끊김

사용자 입력:

> 내 사주를 바탕으로 지금 재물운, 돈 흐름, 수입을 키우는 방향을 먼저 자세히 분석해줘.
> 민준
> 민준
>
> 현철아, 네 사주를 보니 돈 흐름이랑 재물운이 어떤지 아주 명확하게 보여...
> 여기까지만 나오고 멈춰버린다

처리 방향:

- 채팅 스트림 응답이 중간 문장에서 끊기는 증상을 조사한다.
- 화면 렌더링에는 길이 제한이 없어 모델 출력 토큰 한도 가능성을 우선 반영한다.
- 무료/초기 응답의 출력 토큰 예산을 늘리고, `finishReason=length`일 때 서버 로그를 남긴다.
