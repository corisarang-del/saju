# 구글로그인후 사주상담 6회연속실패 제보

## 사용자 입력

구글로그인은 정상화됐지만, 대화를 할 때 사주상담이 6번 연속 실패한다고 제보했다.

## 요청 해석

- Google OAuth 문제와 별개로 `/api/saju/chat` production AI 호출 경로를 진단해야 한다.
- 반복 실패이므로 프롬프트 품질보다 provider 인증, Vercel runtime env, Vertex quota/auth, 저장/RPC 경계를 우선 확인한다.
- 수정 후 실제 production API QA로 무료/유료 첫 상담, 별 차감, 거래 로그를 확인해야 한다.
