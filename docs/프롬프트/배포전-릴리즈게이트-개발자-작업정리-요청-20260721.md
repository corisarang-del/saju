# 배포전 릴리즈게이트 개발자 작업정리 요청

## 사용자 요청
- 배포 전 개발자에게 줄 작업 목록을 전달했다.
- 주요 항목은 Vercel production env 등록, Paddle env gate, Supabase 별 테이블 RLS 잠금, 채팅 별 차감 원자성, 공유 저장소 rate limit, Paddle subscription webhook 검증, `pnpm audit` 취약점 정리다.

## 처리 방향
- 코드 구현이 아니라 개발자 전달용 작업지시 문서로 정리한다.
- 보안 리뷰 기준으로 P0/P1 우선순위, 필수 테스트, 수용 기준, 배포 금지 조건을 포함한다.
- 실제 시크릿 값은 문서에 남기지 않는다.
