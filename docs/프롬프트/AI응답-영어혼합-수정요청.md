# AI 응답 영어 혼합 수정 요청

## 날짜
2026-07-07

## 사용자 요청
사주 상담 답변 중 자미두수와 서양 점성술 설명이 갑자기 영어와 섞여 나오는 문제가 제보됐다.

예시:

```text
또한, 현철 씨는 현재 32~41세 대한(大限, 10년 운세)이 자녀궁(子$). where you are in the current 10-year cycle, it is in the Children's Palace.
The Children's Palace's primary star is Ziwei (紫微, the Emperor Star), which signifies leadership and authority.

서양 점성술(Western Astrology)에서도 현철 씨의 직업운을 엿볼 수 있는데요...
상승궁(ASC, Ascendant)이 양자리(Aries)...
```

## 요구사항
- 사주 상담 답변은 한국어로만 나오게 한다.
- 원문 컨텍스트나 이전 답변에 영어가 섞여 있어도 영어 번역과 영어 병기를 답변에 넣지 않는다.
- 서양 점성술, 자미두수 용어는 한국어로 풀어 설명한다.
