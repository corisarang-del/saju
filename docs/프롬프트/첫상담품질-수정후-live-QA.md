# 첫상담품질 수정후 live QA

사용자가 갱신된 Gemini 첫 상담 5케이스 QA 보고서를 전달했다.

확인 결과:

- 대부분 기준은 통과했다.
- `썸/재회` 케이스에서 `시기(타이밍)` 표현 때문에 `가벼운 외래어 없음`만 확인필요로 남았다.

처리 방향:

- 첫 상담 지시문에 괄호로 외래어를 병기하지 말라는 규칙을 추가한다.
- live QA 스크립트도 괄호 병기 외래어를 명시적으로 잡게 한다.
- 수정 후 Vertex live QA를 다시 실행한다.

추가 확인:

- 외래어 괄호 병기 차단 후 live QA에서 `썸/재회`가 질문으로 끝나지 않는 새 변동 실패가 나왔다.
- 첫 상담은 마지막 문장이 실제 질문 1문장이어야 하고, 설명문이나 조언문으로 끝내지 않는 규칙을 더 강하게 고정한다.
