# 첫채팅 QA 결과 품질보완 요청

## 날짜
2026-07-07

## 사용자 요청
QA 결과로 아래 문제가 전달됐다.

- 유료 첫 채팅 응답이 5문단으로 저장됨. 기준은 1~3문단.
- 유료 첫 채팅 응답에 이모지가 들어감.
- 무료 첫 채팅 응답이 질문으로 끝나지 않음.
- Gemini live QA 6케이스 중 썸/재회는 사주 근거, 번아웃은 구체 행동이 확인필요.

## 처리 방향
- 첫 상담 저장 전 품질 게이트를 추가한다.
- 첫 상담 프롬프트와 Gemini live QA 러너 기준을 강화한다.
- 캐릭터 원본 프롬프트의 이모지 예시를 제거한다.
