# 테스터 피드백 무료베타 실사용 QA 잔여이슈 문서화 요청 20260723

## 사용자 요청

사용자는 테스터 피드백을 개발자에게 전달할 수 있게 정리하고, 지금까지 나온 내용을 문서화하고 메모리에 저장해 달라고 요청했다.

## 원 피드백 요지

- 실제 무료/전체 API QA는 아직 완료하지 못했다.
- Supabase 호스트 `sfpwgywcmhgilrqearsz.supabase.co`가 DNS `ENOTFOUND`로 잡히지 않아 임시 인증 사용자 생성 단계까지 가지 못했다.
- 무료 첫 상담, 별 지급/차감, 거래 로그, assistant 저장 품질은 실제 DB 기준으로 미확인이다.
- 무인증 채팅 API 401 본문이 JSON이 아니라 `Unauthorized` 텍스트라 클라이언트 오류 처리 일관성 확인이 필요하다.
- 무료 베타인데 일부 결제 가능 암시 문구가 남아 있다.
- Playwright 스냅샷에서 `기록을저장하고`처럼 띄어쓰기가 붙어 보인다.

## 정리 방향

- Supabase DNS/API QA 미완료를 무료 베타 배포 blocker로 유지했다.
- 새 이슈 3개를 P1/P2 개발자 액션으로 분리했다.
- 결제모듈은 보존하되 비활성화하는 방향을 유지하면서, 결제 가능 암시 문구는 제거해야 한다고 정리했다.
