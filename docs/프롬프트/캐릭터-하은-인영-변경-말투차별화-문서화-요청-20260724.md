# 캐릭터 하은 인영 변경 말투차별화 문서화 요청 20260724

## 사용자 요청

하은 이름을 인영으로 수정하고 싶다. 그리고 각 상담사들 말투나 화법을 캐릭터마다 더 특색있게 하고 싶은데 생각해봐

지금까지 나온 내용을 개발자가 고칠수있도록 문서화하고 메모리에 저장

## 처리 방향

- PM 역할로 개발자 전달 문서를 작성한다.
- 소스코드는 수정하지 않는다.
- `하은`을 `인영`으로 바꾸되 내부 id `haeun`은 유지하는 방향으로 정리한다.
- 캐릭터별 말투/화법 차별화는 답변 구조, 선호 표현, 금지 표현, QA 기준까지 포함해 정리한다.
- 개발일지와 메모리에 현재 판단을 저장한다.
