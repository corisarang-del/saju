# Paddle 제외 비결제 배포 개발자전달 요청 20260723

## 사용자 요청

사용자는 Paddle 부분은 일단 빼고 배포할 수 있게 개발자에게 전달할 문서를 작성해 달라고 요청했다.

## 포함된 원 요청 요지

- Vercel production env 등록 확인
- Paddle env gate는 기존 배포 차단 요소였지만 이번에는 제외
- Supabase production migration 적용 확인
- 실제 API QA 재실행
- 랜딩 가격 패널, 월간 리포트 가치, 멤버십 운영 표시 보강
- 카피와 접근성 보완
- 배포 전 체크리스트, 실제 API QA 시나리오, CS 플로우, 첫 7일 유저 여정, 핵심 지표 정의

## 정리 방향

- 이번 배포는 “유료 결제 가능 서비스”가 아니라 “무료 상담형 비결제 베타”로 정의했다.
- Paddle env를 단순히 무시하는 것이 아니라, Paddle 의존 UI와 결제 진입점을 feature flag로 막는 방식으로 개발자에게 전달했다.
- Supabase, AI, production origin, rate limit, 실제 API QA는 여전히 출시 전 필수로 유지했다.
