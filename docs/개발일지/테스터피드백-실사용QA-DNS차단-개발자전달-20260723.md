# 테스터 피드백 실사용 QA DNS 차단 개발자전달 20260723

## 작업 내용

- 테스터가 전달한 수정후 실사용 QA 피드백을 개발자 전달 문서로 정리했다.
- 소스코드는 수정하지 않았다.
- 기존 결제모듈 보존형 비활성화 배포 방향과 맞춰, Paddle QA는 제외 가능하지만 Supabase 무료 상담 API QA는 여전히 배포 blocker로 정리했다.

## 핵심 내용

- redirect, Playwright 렌더링, 콘솔 에러 없음은 정상으로 확인됐다.
- Gemini QA는 통과했다.
- 실제 API QA 명령 2개는 모두 `sfpwgywcmhgilrqearsz.supabase.co` DNS `ENOTFOUND`로 중단됐다.
- 무료/유료 첫 상담, 별 차감, 거래 로그, 저장된 assistant 품질은 실제 DB 기준으로 끝까지 확인하지 못했다.
- QA 스크립트가 DNS 실패를 구조화해서 알려주는 개선은 유지할 가치가 있다.
- 관계 고민에서 “몸 상태 기록” 같은 행동 제안은 사람 기준으로 어색할 수 있어 프롬프트/템플릿 보완 여지가 있다.
