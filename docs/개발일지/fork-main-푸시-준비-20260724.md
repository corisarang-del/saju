# fork main 푸시 준비

## 일시
- 2026-07-24 KST

## 배경
- `origin`은 `imgompanda/drsaju-opensource`를 가리키고 있지만 현재 GitHub 계정 `corisarang-del`에는 원본 저장소 write 권한이 없어 push가 403으로 실패했다.

## 작업
- 사용자 요청에 따라 `corisarang-del/saju` fork를 생성한다.
- 로컬 원격 이름은 `fork`로 둔다.
- 최신 로컬 `main`을 `fork/main`으로 푸시한다.

## 주의
- 원본 저장소 `origin`은 유지한다.
- 민감한 env 파일은 이미 커밋 대상에서 제외되어 있다.
