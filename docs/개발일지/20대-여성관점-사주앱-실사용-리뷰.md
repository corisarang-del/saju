# 20대 여성관점 사주앱 실사용 리뷰

- 코드 수정 없이 `localhost:3000/ko`, `/ko/reading`, `/ko/coin-shop` 화면과 관련 컴포넌트 흐름을 확인했다.
- 20대 여성 유저 관점에서는 캐릭터 카드, 2단계 생년월일 입력, 하단 탭 기반 모바일 앱 구조가 호기심 유입에 강점으로 보였다.
- `먼저 챙겨주는 사주친구` 포지션은 매일운, 중요한 선택 전 조언, 관계 고민처럼 20대가 실제로 쓸 만한 맥락과 잘 맞았다.
- 캐릭터 클릭 후 바로 로그인 모달이 뜨는 구조는 가입 전 맛보기 욕구를 충족시키지 못해 전환 장벽으로 보였다.
- `가입시 3별 무료 지급` 문구는 좋지만, 별 3개가 실제로 질문 몇 번인지 즉시 이해하기 어려웠다.
- 비로그인 상태에서 충전샵 가격을 볼 수 없어 결제 전 신뢰 판단이 어려웠다.
- 후기의 `에티오피아 투자까지 맞춤` 같은 강한 표현은 재미보다 광고성으로 느껴질 위험이 있었다.
- 푸터와 환불 FAQ에 placeholder 사업자/이메일 정보가 보이면 실제 서비스 신뢰를 크게 깎을 수 있다고 판단했다.
- 일부 남성 캐릭터의 `어이 동생`, `니 사주` 같은 말투는 20대 여성에게 거칠거나 남성향으로 느껴질 수 있었다.
- 생년월일 입력 화면은 깔끔하지만 년/월/일 필드 라벨, 개인정보 사용/삭제 안내, 쿠키 배너와 하단 CTA 겹침 가능성은 개선 여지가 있었다.
- 우선 개선안은 가입 전 무료 맛보기 결과 노출, 별 사용 기준과 가격 공개, 실제 사업자/문의 정보 정리, 캐릭터 말투 완화, 고민 선택지의 생활 언어화로 정리했다.
- 종합 평가는 `7/10`으로 기록했다. 호기심 유입은 좋지만 결제 전 신뢰와 첫 맛보기 설득이 부족하다는 결론이다.
