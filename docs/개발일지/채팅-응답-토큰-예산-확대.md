# 채팅 응답 토큰 예산 확대

## 배경

민준 캐릭터의 첫 재물운 분석 답변이 `쇠처럼 단`처럼 문장 중간에서 끊기는 문제가 있었다. 채팅 말풍선 렌더링에는 길이 제한이 없고, 서버 채팅 API는 첫 응답 최소 1500자 이상을 요구하면서 무료 응답 `maxOutputTokens`는 1500으로 설정되어 있었다. 모델이 출력 한도에 닿으면 문장이 중간에서 끊길 수 있다.

## 작업

- `src/lib/saju/chat-generation.ts` 추가.
- 무료 채팅 응답 출력 예산을 4000 토큰으로 확대했다.
- 유료/관리자 응답 출력 예산을 10000 토큰으로 확대했다.
- `src/app/api/saju/chat/route.ts`에서 하드코딩된 `1500/8000` 값을 제거하고 `getChatMaxOutputTokens`를 사용했다.
- `finishReason === "length"`일 때 서버 로그에 reading, character, text length, usage를 남기도록 했다.
- `src/lib/saju/chat-generation.test.ts`로 예산 회귀 테스트를 추가했다.

## 검증

- `./node_modules/.bin/vitest run` 통과: 15 files / 36 tests.
- `./node_modules/.bin/eslint src/app/api/saju/chat/route.ts src/lib/saju/chat-generation.ts src/lib/saju/chat-generation.test.ts` 통과.
