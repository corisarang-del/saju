# 가격 코칭 MVP 개발자전달 문서화

- 소스코드는 수정하지 않고 PM/기획자 관점에서 가격/코칭 MVP 완성 계획을 개발자 전달 문서로 정리했다.
- `docs/pm/가격-코칭-MVP-완성-개발자-전달.md`를 새로 작성했다.
- 가격 정책은 `10별 3,900원`, 기존 `30/70/250별`, `월 9,900원 멤버십`, 채팅 1별, 월간 리포트 상세판 3별, 종합 사주 백서 5별로 확정했다.
- 코칭 루프는 첫 상담 후 `CoachingSnapshot`을 만들고 오늘피드, 후속 질문, 월간 전략 리포트, 기억 요약에 재사용하는 방향으로 정리했다.
- 대표 고민 5개별 코칭 템플릿과 수용 기준, 테스트 이름, QA 시나리오를 포함했다.
- 사용자 지시에 따라 코드 구현은 개발자가 진행하고, Codex는 문서 산출물만 작성했다.
