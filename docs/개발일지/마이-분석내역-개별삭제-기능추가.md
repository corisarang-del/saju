# 마이 분석내역 개별삭제 기능추가

- 일시: 2026-07-03 22:00:49 KST

## 수정 내용

- `/ko/my-readings` 분석 목록 카드마다 개별 삭제 버튼을 추가했다.
- 삭제 버튼은 클라이언트 컴포넌트 `DeleteReadingButton`으로 분리했다.
- 삭제 전 `window.confirm`으로 복구 불가 안내를 보여준다.
- 삭제 성공 후 전체 페이지 reload 대신 `router.refresh()`로 마이 목록만 새로고침한다.
- 기존 `deleteReading` server action을 재사용해 로그인/소유자 검증을 유지했다.
- 카드 전체 링크 안에 버튼을 중첩하지 않도록 카드 본문 링크와 삭제 버튼을 분리했다.

## 테스트

- `src/lib/saju/my-readings-delete-regression.test.ts`를 추가했다.
- 확인한 동작:
  - 마이 분석 내역 각 항목에 삭제 버튼이 연결된다.
  - 삭제 버튼이 confirm, server action, refresh 흐름을 사용한다.
  - 삭제 server action이 소유자 검증 후 `saju_readings`를 삭제한다.
