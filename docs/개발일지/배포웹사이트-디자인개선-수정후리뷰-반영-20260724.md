## 변경 내용
- `src/components/saju/landing/CharacterCards.tsx`
  - 캐릭터 캐러셀 자동 이동을 제거했다.
  - mount 시 `scrollLeft = 0`, `activeIndex = 0`으로 첫 진입 위치를 고정했다.
  - pagination 버튼 클릭 시 `scrollTo`와 `activeIndex`를 즉시 맞춰 위치와 active state가 어긋나지 않게 했다.
  - PC `min-width:1024px and max-height:760px` 조건에서 카드 폭을 `252px`로 줄였다.
  - 같은 조건에서 이미지 비율을 `aspect-[4/5]`로 낮추고, 정보 영역 padding/태그 margin/하단 액션 padding을 줄여 CTA가 viewport 안에서 더 여유 있게 보이도록 했다.
- `src/lib/design/design-audit-regression.test.ts`
  - 캐러셀 첫 진입 위치가 사용자 조작 기반으로 고정되는지 테스트를 갱신했다.
  - PC 720px 높이에서 카드 하단 액션 영역 breathing room을 확보하는 회귀 테스트를 추가했다.
