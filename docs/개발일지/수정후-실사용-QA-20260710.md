## 남은 문제

1. 실제 `/api/saju/chat` 유료 첫 응답이 4문단으로 저장돼. 첫 상담 기준은 1~3문단이야.
2. 저장 전 품질 게이트가 있다고 문서화돼 있지만, 이번 실사용 QA에서는 4문단 응답이 저장되고 별도 차감됐어.
3. 유료 응답 샘플에 `MC` 같은 영어 약어가 남아 있어. Gemini 단독 QA에서는 영어 혼합 없음으로 통과했지만, 실제 API 컨텍스트에서는 아직 샐 수 있어.

## 개발자에게 전달할 포인트

- `scripts/qa-gemini-first-consultation.mjs` 단독 QA는 통과했지만, 실제 API는 캐릭터 프롬프트와 사주/자미두수/별자리 컨텍스트를 같이 넣기 때문에 결과가 달라져.
- 저장 전 게이트의 문단 계산 기준이 실제 줄바꿈 형태와 맞는지 확인해야 해.
- 유료 첫 상담에서 4문단 이상이면 저장, 별 차감, `chat_used` 증가가 되면 안 돼.
- 영어 혼합 검출 기준에 `ASC`, `MC` 같은 짧은 약어도 포함해야 해.
