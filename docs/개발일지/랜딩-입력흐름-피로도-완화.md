# 랜딩 입력흐름 피로도 완화

- 일시: 2026-07-01 00:19:39 KST

## 작업 내용

- `design-taste-frontend` 기준을 읽고, PM 문서의 6개 티켓을 랜딩/입력/가격표/캐릭터 톤 회귀 테스트로 먼저 고정했다.
- 캐릭터 카드와 로그인 모달, FAQ, 랜딩 가격 안내의 무료 혜택 문구를 가입 전 관점에서 `3회 무료` 중심으로 통일했다.
- 공용 생년월일 폼, 채팅 입력폼, 랜딩 사이드바 편집폼, 리포트 입력폼의 태어난 시간 선택 영역에 `태어난 시간을 몰라도 분석 가능해. 알면 더 정밀하게 볼 수 있어.` 안내를 추가했다.
- 랜딩 가격표와 별 충전 화면의 배지 영역에 `flex-wrap`과 명확한 `gap`을 적용해 `별 70개인기`처럼 붙어 보이지 않도록 했다.
- 랜딩 가격 표면과 FAQ 배경을 강한 보라/어두운 톤에서 따뜻한 중립색 계열로 낮춰 장시간 읽기 피로도를 줄였다.
- 현우 캐릭터의 설명, quote, 태그, 프롬프트를 `좋은 흐름과 주의할 흐름을 같이 정리하는 차분한 직설` 톤으로 조정했다.
- FAQ 구조화 데이터의 무료 혜택 문구도 `3회 무료 상담`으로 맞췄다.
