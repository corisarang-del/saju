# 가격 코칭 MVP 갭 판단

- 소스코드는 수정하지 않고 PM 관점에서 가격 정책과 코칭형 사주풀이 구현 갭을 점검했다.
- `docs/pm/가격-코칭-MVP-갭-판단.md`에 현재 상태, 아쉬운 점, 추가해야 할 제품 단위, 개발자 전달 문장, 수용 기준을 정리했다.
- 현재 가격은 `1별 = 메시지 1회`, 가입 후 3회 무료, 30별/70별/250별 패키지로 공개되어 있지만 가격 조절과 상품 정책 관점에서는 더 정리할 필요가 있다.
- 현재 첫 상담 답변 품질은 많이 좋아졌지만, 원래 PRD의 오늘피드, 행동카드, 타임라인, 월간 전략 리포트, 대화 기억 요약까지 이어지는 코칭 루프는 아직 덜 닫힌 상태로 판단했다.
- 다음 추가 작업은 가격 정책 확정, 코칭 결과 구조화, 대표 고민별 코칭 템플릿, 오늘피드 MVP, 월간 전략 리포트 최소판, 대화 기억 요약이다.
