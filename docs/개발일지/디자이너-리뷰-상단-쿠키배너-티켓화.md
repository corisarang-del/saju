# 디자이너 리뷰 상단 쿠키배너 티켓화

- 소스코드는 수정하지 않고 PM 관점에서 디자이너 재리뷰 피드백을 개발자 전달 문서로 정리했다.
- `docs/pm/디자이너-리뷰-상단-쿠키배너-개발자-전달.md`에 개선 확인 항목과 남은 P1 이슈를 분리해 작성했다.
- 이전 P0였던 쿠키 배너의 하단 입력/CTA 차단은 해결된 것으로 기록했다.
- 개인정보 링크 locale 유지, 첫 캐릭터 이미지 LCP 경고 해소, `3회 무료` 문구, 접근성 라벨, 가격 배지 분리, 후기 톤 개선도 확인 사항으로 기록했다.
- 남은 핵심은 상단 fixed 쿠키 배너가 모바일 `/ko` H1과 `/ko/reading` 제목을 덮는 문제다.
- 누적 디자인 리스크와 결정 사항을 `.codex/memory/`에 업데이트했다.
