- 랜딩 상단에 가입 전 확인 가능한 무료 맛보기 결과 한 문단을 추가했다.
- 별 사용 기준과 충전 가격을 공통 상수로 분리하고 랜딩, FAQ, 코인샵, 구조화 데이터에서 같은 값을 쓰게 정리했다.
- 환불/문의 이메일을 `corisarang@gmail.com`으로 통일하고 푸터, 환불정책, 약관, 개인정보처리방침의 placeholder를 제거했다.
- 고민 선택지 라벨을 `썸/재회`, `이직/퇴사`, `돈 모으기`, `번아웃`, `친구/가족관계`처럼 생활 언어로 바꿨다.
- 캐릭터 말투에 공통 안전장치를 추가하고 `니`, `어이 동생`, `형이`처럼 하대감이 강한 표현을 실제 노출/호출 단계에서 부드럽게 정제했다.
- 가격/고민 라벨/캐릭터 말투 회귀 테스트를 추가했다.
- 검증: `vitest run`, `tsc --noEmit`, 변경 파일 대상 `eslint` 통과. 전체 `eslint`는 기존 미해결 파일에서 14개 에러가 남아 실패했다.
