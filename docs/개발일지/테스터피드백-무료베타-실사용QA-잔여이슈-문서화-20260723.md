# 테스터 피드백 무료베타 실사용 QA 잔여이슈 문서화 20260723

## 작업 내용

- 테스터가 전달한 무료/전체 API QA 미완료와 새 잔여 이슈를 개발자 전달 문서로 정리했다.
- 소스코드는 수정하지 않았다.
- 결제모듈 보존형 비활성화 배포 방향 안에서, Paddle QA는 제외 가능하지만 Supabase 무료 상담 API QA는 여전히 blocker로 분리했다.

## 핵심 피드백

- 실제 무료/전체 API QA는 Supabase DNS `ENOTFOUND`로 미완료다.
- `sfpwgywcmhgilrqearsz.supabase.co`가 해석되지 않아 임시 인증 사용자 생성 단계까지 못 갔다.
- 무료 첫 상담, 별 지급/차감, 거래 로그, assistant 저장 품질은 실제 DB 기준으로 아직 미확인이다.
- 무인증 채팅 API 401 본문이 JSON이 아니라 `Unauthorized` 텍스트라 클라이언트 오류 처리 일관성 확인이 필요하다.
- 무료 베타인데 일부 결제 가능 암시 문구가 남아 있다.
- Playwright 스냅샷에서 `기록을저장하고`처럼 띄어쓰기 붙음이 보인다.
