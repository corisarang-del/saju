# 20대여성피드백 무료베타 문구일관성 후속수정 20260723

## 배경

20대 여성 유저 관점 피드백에서 무료 베타 배포 승인은 계속 보류가 맞다고 전달됐다. 핵심 blocker는 Supabase DNS `ENOTFOUND`로 실제 무료 상담 API QA가 완료되지 않은 점이고, P1로 무료 베타 상태와 충돌하는 결제/충전 가능 암시 문구가 남아 있다는 지적이 있었다.

직전 수정으로 푸터, JSON-LD, placeholder, paywall의 핵심 문구는 대부분 정리했지만, 정적 검색 기준으로 조건부 결제-on 문구와 일부 리포트/코인샵 CTA에 `별 충전` 계열 문구가 남아 있어 후속 정리를 진행했다.

## 수정 내용

- `src/lib/security/free-beta-qa-regression.test.ts`를 강화해 문서의 문제 예시 문구가 주요 공개 화면 소스에 다시 남지 않도록 고정했다.
- `src/app/[locale]/layout.tsx`의 JSON-LD 결제-on 문구도 `별 충전` 대신 `상담권` 표현으로 낮췄다.
- `src/app/[locale]/coin-shop/page.tsx`의 결제 off 안내 제목을 `추가 상담권은 잠시 준비 중이야`로 바꿨다.
- `src/components/saju/coin-shop/CoinShopClient.tsx`의 결제 off 안내와 섹션 제목을 `추가 상담권`, `상담권 선택`으로 바꿨다.
- `src/components/saju/reports/MonthlyReportUnlockClient.tsx`의 결제-on 부족 CTA를 `상담권 확인하기`로 바꿨다.

## 남은 상태

무료 베타 문구 일관성 P1은 코드상 닫혔다. 하지만 P0인 실제 Supabase 무료 상담 API QA는 여전히 DNS 차단으로 미완료다. 따라서 무료 베타 production 배포 승인은 계속 보류가 맞다.
