# 첫상담 답변품질 개선

- 일시: 2026-07-01 00:49:06 KST

## 작업 내용

- 첫 상담 답변 품질을 별도 회귀 테스트로 분리했다.
- 첫 상담 전용 지침 `getFirstConsultationInstructions`를 추가해 고민 반영, 시간 미상 안심, 1~3문단, 다음 질문, 금지 표현을 구조적으로 고정했다.
- `/api/saju/chat`에서 첫 대화 여부와 태어난 시간 입력 여부를 계산해 첫 상담 전용 지침을 시스템 프롬프트에 주입했다.
- 기존 “답변 끝에 질문 금지” 규칙을 첫 상담에서는 짧은 다음 질문/선택지를 1개 남기도록 수정했다.
- 초기 자동 상담 프롬프트를 `자세히 분석`에서 `차분히 정리` 톤으로 완화하고, 캐릭터별 고민 키워드를 유지했다.
- 캐릭터 카드와 로그인 모달에 `가입하면 3회 무료`, `1별 = 메시지 1회`, 가입 후 가격 확인 가능 문구를 보강했다.
- 로그인 화면 설명에도 `3회 무료 상담`, `1별 = 메시지 1회` 기준을 추가했다.
- 가격 카드에서 별 개수와 배지가 모바일에서 붙어 보이지 않도록 `min-w-0`, `shrink-0`, gap 구조를 보강했다.
