# 20대 여성관점 가격코칭 재리뷰 후속수정

## 배경
- 재리뷰 문서에서 제품 완성도는 높지만 가격 패널, 캐릭터 문구, 접근성 텍스트, 운영 환경값 확인이 남은 과제로 정리됐다.
- 이번 작업은 소스 수정 가능한 후속 요청으로 보고, 작은 회귀 테스트를 먼저 추가한 뒤 구현했다.

## TDD
- `src/lib/saju/characters.test.ts`
  - 지안 카드가 불안 티저 문구를 쓰지 않고 차분한 표현을 유지하는지 확인했다.
  - 서준/도윤 카드가 `커리어는 시기야`, `사업은 시기야` 같은 부자연스러운 문구를 쓰지 않고 `흐름` 중심 문구를 유지하는지 확인했다.
- `src/lib/saju/landing-consultation-trust.test.ts`
  - 랜딩 가격 영역에서 `MONTHLY_MEMBERSHIP`, `월간 멤버십`, `매월` 문구가 노출되는지 확인했다.
  - 사이드바 비로그인 도움말이 보조기술에서 한 문장으로 읽힐 수 있도록 `aria-label`을 갖는지 확인했다.

## 수정 내용
- `src/app/[locale]/page.tsx`
  - 랜딩 가격 패널에 월간 멤버십 한 줄 안내를 추가했다.
  - 월 가격과 매월 지급 별 수량은 `MONTHLY_MEMBERSHIP` 상수를 사용해 가격 정책 source of truth와 연결했다.
- `src/lib/saju/characters.ts`
  - 지안 카드 문구를 불안 유도형 티저에서 차분한 재회 상담 문구로 바꿨다.
  - 서준/도윤 카드 문구를 `시기야` 단정형에서 `흐름`, `방향`, `조건` 중심 표현으로 완화했다.
  - 관련 프롬프트 안의 어색한 조사 표현도 함께 정리했다.
- `src/components/saju/landing/LoginSidebar.tsx`
  - 시각 줄바꿈은 유지하되 `aria-label`로 보조기술이 `로그인하면 모든 대화 기록을 저장하고 다시 볼 수 있어요`를 한 문장으로 읽을 수 있게 했다.

## 검증
- `pnpm exec vitest run src/lib/saju/characters.test.ts src/lib/saju/landing-consultation-trust.test.ts`
  - 2개 파일 / 11개 테스트 통과
- `pnpm test`
  - 50개 파일 / 201개 테스트 통과
- `pnpm lint`
  - 통과
- `pnpm exec tsc --noEmit`
  - 통과
- `pnpm build`
  - Next.js 16.2.11 프로덕션 빌드 통과
- `pnpm test:env`
  - 기본 필수 환경변수 검사 통과
- `pnpm audit --prod --audit-level high`
  - high 이상 취약점 없음, low 1건만 남음
- `REQUIRE_PADDLE_ENV=true pnpm test:env`
  - 실패 확인. Paddle 운영 키/상품/가격 환경변수가 아직 비어 있거나 placeholder라서 운영 배포 전 등록이 필요하다.
