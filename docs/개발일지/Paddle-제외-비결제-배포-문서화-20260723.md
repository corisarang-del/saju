# Paddle 제외 비결제 배포 문서화 20260723

## 작업 내용

- Paddle 결제/멤버십을 이번 배포 범위에서 제외하고 비결제 베타로 배포하기 위한 개발자 전달 문서를 작성했다.
- 소스코드는 수정하지 않았다.
- 기존 P0/P1/P2 항목을 Paddle 제외 배포 기준으로 재정리했다.

## 핵심 결정

- 이번 배포는 무료 상담 중심 비결제 베타다.
- Paddle env gate는 이번 배포 blocker에서 제외한다.
- 대신 결제 진입점, 코인샵, 멤버십, 별 충전, 유료 리포트 unlock은 숨기거나 준비 중으로 막아야 한다.
- Supabase RLS/RPC, 별 예약/환불, rate limit, production origin, 실제 API QA는 여전히 P0다.
