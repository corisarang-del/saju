# 상단 쿠키배너 흐름형 전환

- 일시: 2026-07-01 00:57:11 KST

## 작업 내용

- `design-taste-frontend` 기준으로 상단 fixed overlay보다 콘텐츠 흐름 안의 compact notice가 모바일 정보 위계에 더 적합하다고 판단했다.
- 쿠키 배너를 `fixed top` overlay에서 문서 흐름 안의 `role="region"` notice로 전환했다.
- `ClientWidgets`를 locale layout에서 `{children}`보다 먼저 렌더링해 쿠키 안내가 페이지 첫 영역을 덮지 않고 공간을 차지하게 했다.
- 모바일 safe-area padding은 유지하되, 제목과 입력 컨트롤 위에 떠 있지 않게 했다.
- 개인정보처리방침 링크는 기존 i18n `Link`와 `href="/privacy-policy"`를 유지했다.
- 동의/거부/닫기 상태 저장 정책은 유지하고, 버튼 active feedback을 유지했다.
