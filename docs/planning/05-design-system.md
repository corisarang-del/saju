# 월간사주 디자인 시스템

## 방향
- 화사하고 세련된 프리미엄 모바일 앱
- 신뢰와 위로가 같이 느껴지는 상담 앱
- 캐릭터는 고급 웹툰 상담가풍

## 팔레트
- Background: warm white, pearl, soft sky
- Surface: translucent white, pale ivory
- Primary: deep teal
- Accent: muted gold
- Support: rose beige

## UI 원칙
- 첫 화면은 캐릭터 선택을 중심으로 한다.
- 오늘피드는 카드형이지만 과한 중첩 카드는 피한다.
- 채팅 화면은 캐릭터별 톤과 오늘피드로 자연스럽게 이어진다.
- 모바일 하단 5탭을 기본 내비게이션으로 둔다.
