# 월간사주 화면 구성

## 캐릭터
- 경로: `/[locale]`
- 역할: 사용자가 상담 캐릭터를 선택한다.
- 주요 요소: 캐릭터 카드, 하나 캐릭터 카드, 무료 별 안내, 로그인 유도

## 오늘피드
- 경로: `/[locale]/today`
- 역할: 선택한 캐릭터가 오늘의 흐름을 먼저 알려준다.
- 주요 요소: 캐릭터 인사, 행동카드 3개, 아침/점심/저녁 타임라인, 채팅 CTA, 리포트 CTA

## 채팅
- 경로: `/[locale]/chat/[characterId]`
- 역할: 기억 기반 캐릭터 상담을 제공한다.
- 주요 요소: 대화 목록, 추천 질문, 오행 차트, 무료 할당량 상태, 결제 유도

## 리포트
- 경로: `/[locale]/reports`
- 역할: 월간 전략 리포트와 대화 요약 리포트를 보여준다.
- 주요 요소: 이번 달 전략, 관계/커리어/돈/건강 섹션, 종합 사주 백서 유료 CTA

## 마이
- 경로: `/[locale]/my`
- 역할: 사주 정보, 결제, 모델 설정 노출 여부, 알림 설정을 관리한다.
- 주요 요소: 프로필, 생년월일, 캐릭터별 대화 기록, 구독/단품 결제 상태
