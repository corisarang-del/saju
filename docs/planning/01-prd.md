# 월간사주 PRD

## 프로젝트개요
- 이름: 월간사주
- 한 줄 설명: 먼저 챙겨주는 사주친구이자 대화형 인생운 코치
- 유형: 모바일 우선 웹앱 PWA
- 난이도: 중상 MVP

## 사용자 시나리오
결정피로가 큰 직장인이 아침이나 중요한 선택 직전에 앱을 연다. 사용자는 막연한 운세가 아니라 오늘 해야 할 일, 피해야 할 일, 말투와 타이밍 조언을 받고 싶어한다. 캐릭터는 사용자의 사주, 관심사, 최근 대화 요약을 기억하고 대화가 쌓일수록 조금 더 친근하게 반응한다.

## 핵심기능목록
필수:
- 캐릭터 선택
- 선택한 캐릭터의 오늘 에이전트 피드
- 오늘 행동카드 3개
- 아침/점심/저녁 운세 타임라인
- 기억 기반 캐릭터챗
- 월간 전략 리포트
- 대화 요약 리포트
- 유료 종합 사주 백서
- 소연을 하나로 리브랜딩

선택:
- 고급 푸시 자동화
- 깊은 관계 단계
- 완전 장기기억 검색

## 기술스택
- Next.js, React 19, TypeScript
- Supabase
- Vercel AI SDK
- 관리자 선택형 AI 모델 제공자 어댑터
- Tailwind CSS
- Vitest
- pnpm

## 화면구성
- 캐릭터
- 오늘피드
- 채팅
- 리포트
- 마이

## 상세기능명세
- 오늘피드는 선택한 캐릭터의 말투로 오늘의 핵심 흐름을 먼저 보여준다.
- 행동카드는 `do`, `avoid`, `relationship` 세 종류로 구성한다.
- 타임라인은 `morning`, `afternoon`, `evening` 세 구간으로 구성한다.
- 기억 요약은 사주 프로필, 반복 고민, 최근 대화 요약, 대화 횟수 기반 친근도 톤을 포함한다.
- 무료 할당량을 다 쓰면 단품 결제와 구독 선택지를 모두 제안한다.
- 종합 사주 백서는 유료 아이템 구매 후 접근한다.

## 디자인 가이드
- 화사하고 세련된 배경
- 신뢰와 위로를 주는 고급스러운 톤
- 기존 어두운 보라 중심 팔레트는 줄인다.
- 캐릭터는 고급 웹툰 상담가풍으로 제작한다.

## 제약사항
- pnpm만 사용한다.
- TDD 방식으로 구현한다.
- 외부 AI, 결제, DB는 시스템 경계로 보고 테스트에서 fake 처리한다.
- 의료, 법률, 투자 확정 판단처럼 보이는 표현을 피한다.
